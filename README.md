## 🛠️ Installation and Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/vkk28/Varun-Portfolio.git
   cd Varun-Portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
